# Narwhal .NET library

A .NET standard library to programatically load and save docker volumes from disk

# Getting Started

Install via .NET CLI
```bash
$ dotnet add package narwhal-lib
```

or 

Install via NuGet Package Manager
```powershell
PM> Install-Package Narwhal .NET library
```
